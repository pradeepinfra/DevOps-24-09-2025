## ğŸª£ 2. Basic Concepts

| Term | Description |
|------|--------------|
| **Bucket** | The top-level container where objects (files) are stored. Example: `my-aws-demo-bucket` |
| **Object** | Any file you upload (image, video, log, text file, etc.) |
| **Key** | The name (path) to identify your object inside the bucket |
| **Region** | Buckets are created in a specific AWS region (e.g., `us-east-1`) |
| **Versioning** | Allows you to keep multiple versions of an object |
| **Storage Class** | Defines cost and access speed (Standard, IA, Glacier, etc.) |
| **Access Control** | Controls who can read/write your objects (ACLs, Bucket Policy, IAM) |
